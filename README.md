# YTM

Just another Unofficial YouTube Music Desktop made with Electron, but with 
way less features.

If you want Cool™️  features, take a look at 
https://github.com/ytmdesktop/ytmdesktop.

Most of the code here is based on ytmdesktop.

## Features

- Custom CSS support
- MPRIS integration
- Tray icon
- Discord integration
- YouTube Music
